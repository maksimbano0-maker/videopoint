# 🎬 Halftone Video Effect

Набор Python-скриптов для преобразования обычного видео в стилизованную анимацию из точек (эффект газетной полутоновой печати). В репозитории доступны два варианта: цветной с динамическим дрожанием точек и классический чёрно-белый.

---

## 🛠 Требования

Перед запуском убедитесь, что в вашей системе установлены:

1. **Python 3.x**
2. **FFmpeg** — утилита должна быть добавлена в системный `PATH`, чтобы скрипты могли вызывать её из консоли.

### Зависимости Python
Установите необходимые библиотеки с помощью pip:
```bash
pip install Pillow numpy

Инструкция по использованию1. Цветной полутон (videocolor.py)
Разбивает видео на сетку цветных кружков. Добавляет небольшое случайное смещение (jitter) для создания живого, аналогового и слегка хаотичного эффекта.
 Входной файл по умолчанию: Download.mp4
 Выходной файл: outcolor.mp4

Чёрно-белый полутон (videoblack.py)
Классический эффект типографской печати на белом фоне.
Размер чёрных точек динамически подстраивается под яркость кадра: чем темнее область, тем крупнее точка.
Входной файл: Download.mp4
Выходной файл: output_dots.mp4



Настройка параметров
Вы можете изменить финальный вид видео, отредактировав параметры вызова функций прямо в коде скриптов:
 Параметр Описание dot_size Базовый размер ячейки (шаг сетки в пикселях).
Чем больше число, тем крупнее точки и абстрактнее картинка.
  density Коэффициент плотности точек. Если кружки слишком сильно перекрывают друг друга, уменьшите этот параметр.
 jitter (Только для цветного) Уровень случайного смещения точек по осям X и Y для создания эффекта "дрожания".
fps Количество кадров в секунду для обработки и сборки итогового видео.

 💡 Совет: Если итоговое видео кажется слишком плотным или сливается в кашу, попробуйте немного уменьшить параметр density
(например, до 0.85 или 0.9).

📦 Как это работает внутри
Декомпозиция: Скрипт вызывает ffmpeg для быстрой нарезки исходного видео на отдельные кадры во временную папку temp_frames.
 Проход по пикселям: Кадр анализируется блоками размера dot_size.
 Вычисляется средняя яркость или цвет блока, на основе которых рассчитывается радиус будущей точки.
 Отрисовка: На чистом белом холсте рисуются новые точки.
 Сборка: Из измененных кадров ffmpeg собирает готовый видеофайл с кодеком H.264, после чего временные файлы автоматически удаляются.
